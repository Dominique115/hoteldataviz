Gloria — Views & Critiques

This page embeds two exported dashboards created from the hotel bookings dataset. If an embedded visualization doesn’t load immediately, try refreshing the page or opening the embedded file directly.

View 1: Lead Time Behaviour & Cancellation Dynamics

Overview: This section explores the relationship between lead time (the time between booking and arrival) and cancellation dynamics within the hotel industry. By analyzing this data, hotel managers can identify patterns that may influence booking decisions and cancellation rates.

Embedded file: gloria_View1.html — contains Viz 1.1 (Lead Time density by hotel + cancellations), Viz 1.2 (cancellation rate vs binned lead time with CI), and Viz 1.3 (cancellation heatmap filtered by lead-time brush).

View 2: Guest Segmentation & Service Requirements Analysis

Overview: This view segments hotel guests into meaningful groups based on their travel composition, booking patterns, and service requirements. Understanding these segments helps hotels tailor their services, optimize pricing strategies, and allocate resources effectively to meet guest needs while maximizing revenue.

Embedded file: gloria_view2.html — contains Viz 2.1 (guest stay composition heatmap + special requests bubble chart) and Viz 2.2 (violin/matrix/cancellation linked analysis).

Notes:

  • Keep the three files (gloria_index.html or this file, gloria_view1.html, gloria_view2.html) in the same folder for the iframe embeds to work locally and on GitHub Pages.
  • If you want different iframe heights, edit the CSS variable --frame-height-lg at the top of this document.
